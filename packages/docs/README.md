# @simple-ai/docs

Documentation and component registry site for Simple AI.

This package is private and not published to npm.

## Development

```bash
# From root
npm run docs:dev     # Start dev server on port 4567
npm run docs:build   # Build for production
```

## Structure

- `src/` - Next.js application source
- `public/` - Static assets and registry JSON files
- `project-docs/` - Project documentation
