## Usage

```tsx
import { PromptCrafterNode } from "@/components/react-flow/prompt-crafter-node"
```

The Prompt Crafter Node is a React Flow node component that allows users to craft and edit prompts. It includes:

- Editable text area for prompt creation
- Save and cancel buttons
- Customizable node styling
- Integration with React Flow's node system

```tsx
import { useNodesState } from "reactflow";
import { PromptCrafterNode } from "@/components/react-flow/prompt-crafter-node";

function MyFlow() {
    const [nodes, setNodes] = useNodesState([]);
    
    return (
        <ReactFlow nodes={nodes} onNodesChange={setNodes}>
            <PromptCrafterNode
                id="prompt-1"
                data={{ prompt: "Initial prompt text" }}
                position={{ x: 100, y: 100 }}
            />
        </ReactFlow>
    );
}
```

### Node Properties

The component accepts these standard React Flow node properties:

- `id`: Unique identifier for the node
- `data`: Object containing the prompt text
- `position`: X and Y coordinates for node placement
- `selected`: Boolean indicating if node is selected
- `type`: Node type (default: 'promptCrafter')
