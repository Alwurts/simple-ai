## Usage

```tsx
import { EditableHandle } from "@/components/ui/editable-handle"
```

The Editable Handle component is a customizable node handle for React Flow that allows users to:

- Edit the handle label directly
- Customize handle position and type
- Add custom styling
- Integrate with React Flow's node system

```tsx
import { useState } from "react";
import { EditableHandle } from "@/components/ui/editable-handle";

function MyComponent() {
    const [label, setLabel] = useState("Input");
    return (
        <EditableHandle
            id="input-1"
            type="target"
            position="left"
            label={label}
            onLabelChange={setLabel}
        />
    );
}
```

### Handle Types

The component supports these handle types:

- `source` - Output handle
- `target` - Input handle
- `custom` - Custom handle type

### Positioning

You can position the handle using these positions:

- `top`
- `bottom`
- `left`
- `right`
