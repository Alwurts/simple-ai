## Overview

The Editable Handle component is built on top of React Flow's [Base Handle](https://reactflow.dev/components/handles/base-handle) component, extending its functionality to provide an interactive way to manage handles in your nodes. It allows users to create, edit, and delete handles dynamically, with support for labels and optional descriptions.

## Usage

The Editable Handle system consists of two main components:

1. `EditableHandleDialog`: A popup dialog component for creating new handles
2. `EditableHandle`: The handle component itself that displays and allows editing of existing handles

### Creating New Handles

The `EditableHandleDialog` component is used to create new handles. It provides a popup interface where users can input the handle's label and optional description.

```tsx
import { EditableHandleDialog } from "@/components/ui/flow/editable-handle"

function MyNode() {
  const handleCreate = (name: string, description?: string) => {
    // Add the new handle to your state
    return true // Return true if handle was created successfully
  }

  return (
    <EditableHandleDialog
      variant="create"
      onSave={handleCreate}
      align="start"
    >
      <Button variant="outline" size="sm">
        Create New Handle
      </Button>
    </EditableHandleDialog>
  )
}
```

### Managing Existing Handles

The `EditableHandle` component is used to display and manage existing handles. It provides functionality for editing and deleting handles.

```tsx
import { EditableHandle } from "@/components/ui/flow/editable-handle"
import { Position } from "@xyflow/react"

function MyNode({ id }: NodeProps) {
  const handleUpdate = (handleId: string, newName: string, newDescription?: string) => {
    // Update your handle state
    return true // Return true if update was successful
  }

  const handleDelete = (handleId: string) => {
    // Remove the handle from your state
  }

  return (
    <EditableHandle
      nodeId={id}
      handleId="handle-1"
      name="Input 1"
      description="Optional description"
      type="target"
      position={Position.Left}
      onUpdateTool={handleUpdate}
      onDelete={handleDelete}
      showDescription
    />
  )
}
```

## Props

### EditableHandle

| Prop | Type | Required | Default | Description |
| --- | --- | --- | --- | --- |
| nodeId | string | Yes | - | The ID of the node this handle belongs to |
| handleId | string | Yes | - | Unique identifier for the handle |
| name | string | Yes | - | The label displayed next to the handle |
| description | string | No | - | Optional description text |
| type | 'source' \| 'target' | Yes | - | The type of handle (input or output) |
| position | Position | Yes | - | The position of the handle (left, right, top, bottom) |
| onUpdateTool | (handleId: string, newName: string, newDescription?: string) => boolean | Yes | - | Callback when handle is updated |
| onDelete | (handleId: string) => void | Yes | - | Callback when handle is deleted |
| showDescription | boolean | No | false | Whether to show the description field |

### EditableHandleDialog

| Prop | Type | Required | Default | Description |
| --- | --- | --- | --- | --- |
| variant | 'edit' \| 'create' | Yes | - | Whether this is for creating a new handle or editing an existing one |
| onSave | (newLabel: string, newDescription?: string) => boolean | Yes | - | Callback when the form is saved |
| label | string | No | - | Initial label value (for edit mode) |
| description | string | No | - | Initial description value (for edit mode) |
| align | 'start' \| 'end' | No | 'start' | Alignment of the popup dialog |
| showDescription | boolean | No | false | Whether to show the description field |
| children | ReactNode | Yes | - | The trigger element for the popover |
