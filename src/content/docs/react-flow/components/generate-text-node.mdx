## Usage

```tsx
import { GenerateTextNode } from "@/components/ui/generate-text-node"
```

The Generate Text Node component is a React Flow node that allows users to generate text content. It includes:

- Editable text input field
- Generate button with loading state
- Customizable node styling
- Built-in error handling

```tsx
import { useState } from "react";
import { GenerateTextNode } from "@/components/ui/generate-text-node";

function MyFlowComponent() {
    const [text, setText] = useState<string>("");
    return (
        <GenerateTextNode
            id="text-node-1"
            data={{ text }}
            onTextChange={(newText) => setText(newText)}
            onGenerate={() => {
                // Handle text generation logic
            }}
        />
    );
}
```

### Node Properties

The component accepts these props:

- `id: string` - Unique identifier for the node
- `data: { text: string }` - Current text content
- `onTextChange: (text: string) => void` - Callback when text changes
- `onGenerate: () => void` - Callback when generate button is clicked
- `isGenerating?: boolean` - Loading state for the generate button
