## Usage

```tsx
import { StatusEdge } from "@/components/react-flow/status-edge"
```

The Status Edge component is a custom edge component for React Flow that includes:

- Different status states (success, error, warning, processing)
- Animated progress indicator
- Customizable colors and styles
- Built-in edge types and markers

```tsx
import { useEdgesState } from "reactflow";
import { StatusEdge } from "@/components/react-flow/status-edge";

function MyFlowComponent() {
    const [edges, setEdges] = useEdgesState([]);
    return (
        <ReactFlow
            edges={edges}
            edgeTypes={{
                status: StatusEdge
            }}
        />
    );
}
```

### Available Statuses

The component includes these predefined status states:

- `success` - Green color with checkmark
- `error` - Red color with cross
- `warning` - Yellow color with exclamation
- `processing` - Blue color with animated spinner

Each status has a corresponding visual indicator that appears along the edge.
