## Overview

The Visualize Text Node is a React Flow node component that renders text content with Markdown support. It provides a clean, resizable interface for displaying formatted text from other nodes in your flow, making it perfect for visualizing outputs or documentation.

## Usage

The Visualize Text Node provides a resizable container for displaying formatted text content. Here's how to use it in your React Flow application:

```tsx
import { VisualizeTextNode } from "@/components/ui/flow/visualize-text-node"

const nodes = [
  {
    id: "visualize-1",
    type: "visualize-text",
    position: { x: 0, y: 0 },
    data: {
      input: "### Markdown Support\n\nThis node supports:\n- Headers\n- Lists\n- And other markdown features",
      status: "success"
    }
  }
]

function MyFlowComponent() {
  return (
    <ReactFlow
      nodes={nodes}
      nodeTypes={{ "visualize-text": VisualizeTextNode }}
    />
  )
}
```

### Features

The node includes several key features:

1. **Markdown Support**
   - Renders markdown-formatted text
   - Supports headers, lists, and other markdown syntax
   - Clean and readable output

2. **Resizable Interface**
   - Drag to resize the node
   - Scrollable content area
   - Text selection enabled

3. **Input Handle**
   - Single input handle on the left side
   - Accepts text input from connected nodes

4. **Status Indication**
   - Visual status feedback (processing, error, success, idle)
   - Status-based border highlighting

## Props

### Node Data Structure

| Property | Type | Description |
| --- | --- | --- |
| status | 'processing' \| 'error' \| 'success' \| 'idle' \| undefined | Current status of the node |
| input | string \| undefined | The text content to be displayed |

### Component Props

| Prop | Type | Required | Default | Description |
| --- | --- | --- | --- | --- |
| id | string | Yes | - | Unique identifier for the node |
| data | "" | Yes | - | Node data containing status and input text |
| selected | boolean | No | - | Whether the node is selected |
| onDeleteNode | => void | No | - | Callback when the delete button is clicked |
