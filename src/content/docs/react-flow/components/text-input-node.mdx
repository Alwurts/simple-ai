## Usage

```tsx
import { TextInputNode } from "@/components/ui/text-input-node"
```

The Text Input Node component is a specialized node for React Flow that allows users to input and edit text. It includes:

- Built-in text input functionality
- Customizable placeholder text
- Value and change handlers
- Integration with React Flow's node system

```tsx
import { useState } from "react";
import { TextInputNode } from "@/components/ui/text-input-node";

function MyFlowComponent() {
    const [text, setText] = useState<string>("");
    return (
        <TextInputNode
            id="text-node-1"
            data={{ text }}
            onTextChange={(newText) => setText(newText)}
        />
    );
}
```

### Customization

You can customize the text input node with these props:

- `placeholder`: Set custom placeholder text
- `className`: Add custom styles using Tailwind CSS
- `disabled`: Disable the text input
- `onTextChange`: Handle text changes
