## Overview

The Text Input Node is a React Flow node component that provides a resizable textarea for inputting and editing text. It features a clean interface with a single output handle, making it perfect for creating text content that can be processed by other nodes in your flow.

## Usage

The Text Input Node provides a resizable textarea for entering and editing text. Here's how to use it in your React Flow application:

```tsx
import { TextInputNode } from "@/components/ui/flow/text-input-node"
import { useState } from "react"

function MyFlowComponent() {
  const [value, setValue] = useState("Initial text")

  return (
    <ReactFlow
      nodes={[{
        id: "input-1",
        type: "text-input",
        position: { x: 0, y: 0 },
        data: {
          status: "idle",
          config: { value }
        }
      }]}
      nodeTypes={{ "text-input": TextInputNode }}
      onTextChange={setValue}
    />
  )
}
```

### Features

The node includes several key features:

1. **Resizable Interface**
   - Drag to resize the node
   - Textarea automatically fills available space
   - Clean and intuitive design

2. **Text Editing**
   - Multi-line text input support
   - Placeholder text
   - Auto-resize with content

3. **Output Handle**
   - Single output handle on the right side
   - Sends text content to connected nodes

4. **Status Indication**
   - Visual status feedback (processing, error, success, idle)
   - Status-based border highlighting

## Props

### Node Data Structure

| Property | Type | Description |
| --- | --- | --- |
| status | 'processing' \| 'error' \| 'success' \| 'idle' \| undefined | Current status of the node |
| config | '' | Configuration object containing the textarea value |

### Component Props

| Prop | Type | Required | Default | Description |
| --- | --- | --- | --- | --- |
| id | string | Yes | - | Unique identifier for the node |
| data | TextInputData | Yes | - | Node data containing status and config |
| selected | boolean | No | - | Whether the node is selected |
| deletable | boolean | No | - | Whether the node can be deleted |
| onTextChange | (value: string) => void | No | - | Callback when text content changes |
| onDeleteNode | () => void | No | - | Callback when the delete button is clicked |
